# HomeEssentials - Full-Stack E-commerce Platform

A minimalist and professional e-commerce web application specialized in kitchen appliances and home decor. Built with a focus on clean UI and robust backend logic.

## Key Features
- **Modern UI:** Soft UI aesthetic using Bootstrap and custom CSS.
- **Admin Dashboard:** Manage orders, stock, and return requests.
- **Wallet & Refund:** Automatic wallet credits for returned items.
- **Email System:** Order status notifications via Gmail SMTP.

##  Tech Stack
- **Backend:** Python (Django)
- **Frontend:** HTML5, CSS, JavaScript, Bootstrap
- **Database:** SQLite
